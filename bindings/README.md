# Bindings

Language bindings for the Tensora Rust library.

## Available Bindings

| Directory | Language | Package |
|-----------|----------|---------|
| [`python/`](python/) | Python | `tensora_py` |

## Python

The Python package (`tensora_py`) provides:
- PyTorch integration for SafeTensors and ServerlessLLM loading
- TensorFlow convenience API
- vLLM custom weight loader
- pytest-benchmark performance suites

See [`python/README.md`](python/README.md) for setup and usage.
